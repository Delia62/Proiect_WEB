Exporta baza de date

Exportarea datelor dintr-o bază de date PostgreSQL folosind pg_dump este o metodă eficientă și versatilă pentru a crea o copie de siguranță a bazei de date sau pentru a transfera datele între servere. Utilitarul pg_dump este conceput pentru a salva conținutul unei baze de date PostgreSQL într-un fișier, care poate fi apoi restaurat folosind pg_restore sau comenzi SQL standard.

Baza de date este exportată într-un fișier cu extensia .sql, care conține instrucțiuni SQL pentru a recrea baza de date. Acest fișier poate fi apoi importat într-o altă bază de date PostgreSQL folosind utilitarul pg_restore sau comenzi SQL standard.

Daca se doreste exportea in alt format, va rog accesati pagina de export in format CSV.

Pentru a exporta baza de date, apasați pe butonul de mai jos.
